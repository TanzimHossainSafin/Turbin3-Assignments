## Getting Started

### Prerequisites

Ensure you have the following installed:

- [Rust](https://www.rust-lang.org/tools/install) (latest stable)
- [Solana CLI](https://docs.solana.com/cli/install-solana-cli-tools)
- [Anchor Version Manager](https://www.anchor-lang.com/docs/installation)
- [Node.js 16+](https://nodejs.org/) & Yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/TanzimHossainSafin/Turbin3-Assignments.git
   cd Turbin3-Assignments
   ```

2. **Navigate to a specific assignment**
   ```bash
   cd assignment-0
   ```

3. **Build the project**
   ```bash
   anchor build
   ```

4. **Run tests**
   ```bash
   anchor test
   ```
